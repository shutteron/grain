# GRAIN 作例写真の設計プラン

## 概要

GRAINを「写真教材らしく」するために、初心者が文章だけでは理解しにくい内容を「見て分かる」ようにするための作例写真の撮影プランです。

このドキュメントは今後追加する作例写真の仕様書であり、実際の画像ファイルはまだ追加されていません。

---

## 作例写真を入れる目的

作例写真の役割：
- **視覚的な理解**: 文章で説明した概念を「見て確認」する
- **比較**: Before/After により、設定変更の効果を明確に示す
- **初心者の不安を減らす**:「自分も同じような失敗をする可能性がある」と気づき、解決策を学ぶ

優先して説明したい内容：
1. 写真が暗い理由と、その改善方法
2. 背景がボケる/ボケない条件の違い
3. ピントが合う/合わない違いと、その見分け方
4. 逆光で顔が暗くなる現象と、その対策
5. ISOを上げたときの明るさとザラつきのトレードオフ

---

## 優先して作例写真を入れる5教材

以下の5教材は、初心者が実際に撮影して失敗しやすい内容であり、写真での説明が特に有効です。

## 教材1: dark-photo-trouble

### 教材ID
`dark-photo-trouble`

### 教材タイトル
写真が暗く見える失敗

### 作例の目的

「写真が暗い」という失敗は、光が足りていない状態です。同じシーンを異なる明るさで撮影した写真を並べることで、以下を理解させます：

- 暗い写真は「失敗」ではなく「光の不足」
- 明るさを調整することで印象が大きく変わる
- 同じシーンでも、自分好みの明るさを選ぶ重要性

### 必要な写真

| 用途 | 説明 |
|------|------|
| **暗すぎる写真** (dark) | 露出補正 -1.5EV 程度。見えづらく、沈んだ印象 |
| **ちょうどよい写真** (good) | 露出補正 0EV～0.5EV。被写体と背景のバランスがとれた状態 |
| **明るすぎる写真** (bright) | 露出補正 +1.5EV 程度。明るいが、やや眠い印象になる可能性 |

### 撮影メモ

- **場所**: 日中で光が十分ある場所（屋外、明るい室内）
- **被写体**: わかりやすい風景や静物。料理、花、カフェの風景など、初心者にとって身近なもの
- **光**: できるだけ均等な光。フラッシュは使わない（自然光を活用）
- **フレーミング**: 3パターン全て同じアングル、同じフレーミングで撮影
- **注意点**: 
  - デジタルズーム、トリミングを避ける（同じレンズ・距離で撮影）
  - 3枚の写真で「光の足りない→ちょうどいい→光が多すぎる」という明るさの流れを表現

### 画像ファイル名案

```
/public/images/examples/dark-photo-trouble-dark.jpg
/public/images/examples/dark-photo-trouble-good.jpg
/public/images/examples/dark-photo-trouble-bright.jpg
```

### 教材ページで表示する説明文（案）

| パターン | 説明文 |
|---------|--------|
| dark | **暗すぎる写真**: 光が足りていない状態。被写体が見えづらく、全体が沈んだ印象になります |
| good | **ちょうどよい写真**: 被写体と背景のバランスがとれた状態。初心者は、この「ちょうどいい」を探す練習が大事です |
| bright | **明るすぎる写真**: 光が多すぎる状態。明るいですが、やや眠い印象になることもあります |

## 教材2: background-blur-trouble

### 教材ID
`background-blur-trouble`

### 教材タイトル
背景がボケない失敗

### 作例の目的

背景ボケは F 値だけで決まらない、ということを理解させます。以下の3つの写真を並べることで、背景ボケに影響する要因（距離、背景の遠さ）を視覚的に示します：

- 被写体に近づくと、背景がボケやすくなる
- 背景を遠ざけると、ボケやすくなる
- F 値と距離の両方で、ボケ具合が変わる

### 必要な写真

| 用途 | 説明 |
|------|------|
| **ボケていない写真** (no-blur) | 被写体が遠く、背景も近い状態。全体にピントが合ったような見え方 |
| **被写体に近づいた写真** (close) | 同じ被写体に近づいて撮影。背景が明らかにボケています |
| **背景を遠ざけた写真** (distance) | 被写体の位置は変わらず、背景を遠ざけて撮影。ボケが強調されます |

### 撮影メモ

- **場所**: 背景に奥行きがあるロケーション（公園、街並み、室内の奥にポスターなど）
- **被写体**: 人物、花、小物など、わかりやすいもの
- **光**: 均等な光。午前中の柔らかい光が理想的
- **カメラ設定**: 
  - 同じレンズを使用（35mm か 50mm が目安）
  - F 値は同じに固定（F2.8～F4.0 程度）
  - ピントは被写体に合わせる
- **フレーミング**: 
  - 写真1：被写体が小さく見える距離（遠い）
  - 写真2：被写体に顔が見える距離まで近づく
  - 写真3：被写体の距離は写真1と同じだが、背景が遠ざかった状態
- **注意点**: 
  - 露出が大きく変わらないよう注意
  - 3パターンで背景ボケの違いが「明確に」見えることが重要

### 画像ファイル名案

```
/public/images/examples/background-blur-trouble-no-blur.jpg
/public/images/examples/background-blur-trouble-close.jpg
/public/images/examples/background-blur-trouble-distance.jpg
```

### 教材ページで表示する説明文（案）

| パターン | 説明文 |
|---------|--------|
| no-blur | **背景がボケていない状態**: 被写体が遠く、背景も近い状態です。背景も含めて全体が見えますが、主役が目立ちません |
| close | **被写体に近づいた写真**: 同じレンズでも被写体に近づくと、背景がボケやすくなります。主役が引き立ちます |
| distance | **背景を遠ざけた写真**: 被写体の距離は変わらず、背景を遠ざけると、さらに背景がボケます。ボケの強さは、距離とレンズの組み合わせで決まります |

## 教材3: focus-trouble

### 教材ID
`focus-trouble`

### 教材タイトル
ピントが合わない失敗

### 作例の目的

カメラが「自分の意図した場所」にピントを合わせるとは限らない、という現実を示します。正しいピント位置と、失敗例を並べることで、以下を理解させます：

- 人物撮影では、目や顔にピントが合うことが大事
- カメラの顔認識が働かない場合もある
- ピント位置を確認する習慣の重要性

### 必要な写真

| 用途 | 説明 |
|------|------|
| **ピント正解** (good) | 顔、特に目にピントが合った写真。被写体がシャープで見やすい |
| **背景にピントが抜けた写真** (background) | 被写体の顔は柔らかく、背景や奥の景色にピントが合っている |
| **手前のものにピントが合った写真** (foreground) | カメラが誤認識して、被写体より手前のものにピントを合わせた例 |

### 撮影メモ

- **場所**: 人物が撮影しやすい室内環境（窓の光がある場所が理想的）
- **被写体**: 人物（顔がはっきり見える距離）
- **光**: 窓からの自然光が理想的。逆光でも順光でもよい
- **カメラ設定**:
  - 被写体に近づいて撮影（最低 1m 程度の距離）
  - F 値は F2.8 程度でボケを作る
  - ピントポイントを意識的に変える
- **フレーミング**: 
  - 写真1：目にピントが合った状態。カメラの AF を顔認識で行う
  - 写真2：カメラを背景に向けて AF してから被写体に戻す（背景にピント）
  - 写真3：被写体の手前にピントポイントを設定（例：眼鏡、フォーク、手など）
- **注意点**: 
  - ピントの違いが明確に見えることが最優先
  - 露出は 3 パターン同じに統一

### 画像ファイル名案

```
/public/images/examples/focus-trouble-good.jpg
/public/images/examples/focus-trouble-background.jpg
/public/images/examples/focus-trouble-foreground.jpg
```

### 教材ページで表示する説明文（案）

| パターン | 説明文 |
|---------|--------|
| good | **ピント正解**: 顔、特に目にピントが合った状態。被写体がシャープに見え、視線が自然に向かいます |
| background | **背景にピント抜け**: カメラが被写体ではなく背景にピントを合わせた失敗例。被写体がやや柔らかく見えます |
| foreground | **手前にピント**: カメラが被写体より手前のものに反応して、ピントを合わせてしまった例。背景にも被写体にも、ピントが合っていません |

## 教材4: dark-face-trouble

### 教材ID
`dark-face-trouble`

### 教材タイトル
顔が暗くなる失敗

### 作例の目的

背景（例：窓の外）が明るいと、カメラが背景に合わせて露出を決め、顔が暗くなる現象を示します。以下を理解させます：

- 逆光では顔が暗くなりやすい
- 露出補正で顔を明るくする方法
- レフ板やストロボで光を足す方法

### 必要な写真

| 用途 | 説明 |
|------|------|
| **逆光で顔が暗い写真** (backlight) | 窓を背にした人物。背景の空は明るいが、顔は暗い状態 |
| **露出補正で明るくした写真** (exposure) | 同じシーン、同じアングルで露出補正 +1.0 EV～+1.5 EV で撮影。顔が明るくなる |
| **光を足した写真** (light-added) | レフ板またはストロボで顔に光を足した状態。自然な明るさ |

### 撮影メモ

- **場所**: 窓のある室内（逆光条件を作る）
- **被写体**: 人物。顔の表情が見える距離
- **光**: 
  - 背景：窓の外の明るい空
  - 被写体：室内光のみ（最初は光を足さない）
- **カメラ設定**:
  - AF は人物の顔に向ける
  - 露出補正は段階的に変える（-0.5 → 0 → +1.0 EV）
- **フレーミング**: 
  - 写真1：露出補正なし。背景の空を優先した自動露出
  - 写真2：露出補正 +1.0～+1.5 EV。顔が明るくなる
  - 写真3：レフ板（白い紙でも可）で顔に光を反射させるか、ストロボで軽く補光
- **注意点**: 
  - 3 パターンで顔の明るさの違いが「はっきり」見えることが重要
  - ストロボを使う場合は、自然な光になるよう調整（全光でなく補光）

### 画像ファイル名案

```
/public/images/examples/dark-face-trouble-backlight.jpg
/public/images/examples/dark-face-trouble-exposure.jpg
/public/images/examples/dark-face-trouble-light-added.jpg
```

### 教材ページで表示する説明文（案）

| パターン | 説明文 |
|---------|--------|
| backlight | **逆光で顔が暗い**: 背景の空が明るいため、カメラが背景に合わせて露出を決め、顔が暗くなります |
| exposure | **露出補正で明るく**: 同じシーンで露出補正 +1.0EV～+1.5EV にすると、顔が明るくなります。背景が少し明るくなりすぎる可能性もありますが、顔が優先です |
| light-added | **光を足した写真**: レフ板やストロボで光を足すと、より自然な明るさになります。背景と顔の両立が可能になります |

## 教材5: iso-basic

### 教材ID
`iso-basic`

### 教材タイトル
ISO感度とは

### 作例の目的

ISO を上げるとの明るくなるが、ザラつき（ノイズ）が出やすくなる、というトレードオフを視覚的に示します。以下を理解させます：

- ISO を上げると、暗い場所での写真が明るくなる
- ISO を上げると、画面がザラザラに見えやすくなる
- しかし、ブレた写真より、多少ザラついても撮れている写真の方が良い場面もある

### 必要な写真

| 用途 | 説明 |
|------|------|
| **ISO 低めの暗い写真** (low) | ISO 100～200。暗くて見えづらいが、ザラつきは少ない |
| **ISO 中くらいの写真** (middle) | ISO 800～1600。ちょうどよい明るさ。ザラつきはわずか |
| **ISO 高めの明るい写真** (high) | ISO 3200～6400。明るいが、画面全体が少しザラザラしている |

### 撮影メモ

- **場所**: 薄暗い室内（例：カフェの奥、ライティングが限定的な場所）
- **被写体**: わかりやすい静物（カップ、植物、本など）
- **光**: 室内の窓光のみ。フラッシュは使わない
- **カメラ設定**:
  - F 値：F2.8～F4.0 で固定
  - シャッタースピード：各 ISO で適切に（例：1/30～1/125）
  - 同じレンズ、同じアングルで撮影
- **フレーミング**: 
  - 写真1：ISO 100。暗いが、粒子感は少ない
  - 写真2：ISO 1600。ちょうどよい明るさ
  - 写真3：ISO 6400。明るいが、ザラザラが目立つ（画面全体で確認できる）
- **注意点**: 
  - ザラつきが「見える」ことが最優先。小さい画面だと ザラつきが見えにくいため、十分明るい条件で撮影することも検討
  - 被写体の細部（テクスチャー）がザラつきで失われるのが「見える」こと
  - 3 パターンで露出補正をせず、ISO だけを変えて撮影

### 画像ファイル名案

```
/public/images/examples/iso-basic-low.jpg
/public/images/examples/iso-basic-middle.jpg
/public/images/examples/iso-basic-high.jpg
```

### 教材ページで表示する説明文（案）

| パターン | 説明文 |
|---------|--------|
| low | **ISO 低め**: 室内で ISO 100 で撮影。暗くて見えづらいですが、ザラザラはありません。ISOを上げずにこのまま撮ると、ブレてしまう可能性があります |
| middle | **ISO 中くらい**: ISO 1600 で撮影。適切な明るさで、ザラつきもわずかです。初心者はこのレベルから始めるのがおすすめです |
| high | **ISO 高め**: ISO 6400 で撮影。明るいですが、画面全体が少しザラザラしています。ブレて使えない写真より、ザラザラでも撮れている写真を優先する判断が時には大事です |

## 撮影時の技術的なポイント

### 共通注意事項

1. **解像度**: 将来的には Web 用に最適化（例：1920px 幅、80% 品質の JPEG）
2. **色温度**: 各シリーズで統一感を出す。色温度は可能な限り同じ（同じ時間帯、同じ場所で撮影）
3. **メタデータ**: 撮影設定（ISO, F 値, シャッタースピード, 焦点距離）をメモしておく
4. **著作権**: GRAINでの利用を想定した被写体選定（人物の場合は配慮が必要）
5. **著作権表記**: 必要に応じて、教材ページに撮影者の表記を追加
